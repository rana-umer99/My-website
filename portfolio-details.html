Web Application in Vuejs, Vuetify, PHP Laravel

This web application has following key features:

  • Markup built from scratch using Twitter Bootstrap, HTML, CSS, SASS, Dashkit
  • All of the components are built in Vuejs framework
  • Used Vuetify plugin with PHP Laravel at the backend of the platform
  • Live deal tickers
  • Dashboard page to show latest earnings and spendings
  • Shipment section to manage and view all of your shipments
  • Profile module for all payment, security and address related fields
  • Whats new section to show latest updates on the platform
  • Notification center
  • Bulk buy center
  • Proposed layout design sugesstions using wireframes and get approval from the client
  • Live chat and support groups
  • Quantity allotment modules
